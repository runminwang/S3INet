# S3INet
S3INet: Semantic-information Space Sharing Interaction Network for Arbitrary Shape Text Detection
Arbitrary shape text detection is a challenging task due to significant variations in text shapes, sizes, and aspect ratios. Traditional approaches relying on feature maps generated through the feature pyramid network have limitations in harnessing high-level semantic information and expressing multi-scale features. How to make the obtained feature maps effectively represent text features and facilitate the subsequent processing to suppress false positives is a core issue in text detection techniques. To address these dilemmas, this paper introduces a novel arbitrary shape scene text detector called the Semantic-information Space Sharing Interaction Network (S3INet). The proposed network leverages the Semantic-information Space Sharing Module (S3M) to generate a single-level feature map capable of expressing multi-scale features with rich semantic and prominent foreground, and we propose the Multi-branch Parallel Asymmetric Convolution Module (MPACM) to represent text features, facilitating subsequent text detection processing. Extensive experiments evaluations on five publicly available natural scene text datasets, i.e., CTW-1500, Total-Text, MSRA-TD500, ICDAR2015, ICDAR2017-MLT, and two traffic text datasets, i.e., CTST-1600, and TPD, demonstrate the superiority of our method.

We will make the code publicly available after the paper is published.
